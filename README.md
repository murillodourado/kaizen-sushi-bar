# 🍣 Kaizen Sushi Bar

A responsive landing page for a Japanese restaurant built with HTML, CSS and jQuery.

---

## 📋 About

Kaizen Sushi Bar is a fictional Japanese restaurant located in Rua Galvão Bueno, Liberdade — São Paulo, SP. This project is a fully responsive landing page featuring a hero section, menu, testimonials and footer.

---

## 🚀 Features

- **Sticky header** with smooth scroll spy that highlights the active nav item
- **Responsive navbar** with animated hamburger menu
- **Hero section** with floating image, decorative ring and background shape
- **Menu section** with dish cards, star ratings and order button
- **Testimonials section** with chef spotlight and customer reviews
- **Footer** with restaurant address, hours and contact info
- Fully responsive for **mobile, tablet and desktop**

## 📦 Getting Started

No build tools or dependencies needed — just open the project in your browser.

```bash
# Clone the repository
git clone https://github.com/your-username/kaizen-sushi-bar.git

# Open in browser
open index.html
```

Or use a local server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) on VS Code for the best experience.

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| `> 1025px` | Desktop — full layout, 4 menu cards per row |
| `601px – 1024px` | Tablet — 2 menu cards per row, stacked sections |
| `< 600px` | Mobile — 1 card per row, hamburger menu |
